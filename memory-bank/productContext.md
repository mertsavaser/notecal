## Product Context

- **Problem**: Users need a lightweight way to manage notes alongside calendar events without juggling multiple apps.
- **Solution**: NoteCal offers a unified interface where authentication, onboarding, and profile setup are handled via Firebase, then users land in a focused home experience.
- **User experience goals**:
  - Fast startup with minimal friction.
  - Robust authentication and profile flows that “just work”.
  - Clear separation between onboarding, auth, and main app content.
